# Contact Management System

A modern contact management system built with the MERN stack (MongoDB, Express, React, Node.js). This application allows users to register, log in, and manage their contacts with full CRUD (Create, Read, Update, Delete) operations in a secure, user-friendly interface.

## Features

- **User Authentication**: Secure login and registration with JWT (JSON Web Token) for session management and bcrypt for password hashing.
- **CRUD Operations**: Create, view, edit, and delete contacts.
- **Responsive UI**: Built with React for a modern and user-friendly interface.
- **Form Validation**: Express Validator for validating data on the backend.
- **Loading and Notifications**: React Spinner for loading states and React Toastify for success/error notifications.
- **Interactive Popups**: SweetAlert2 used for delete confirmation dialogs.
- **Data Management**: Context API for efficient data handling.

## Technologies Used

- **Frontend**: React, React Router, React Spinner, React Toastify, SweetAlert2
- **Backend**: Node.js, Express, MongoDB
- **Security**: JWT, bcrypt
- **Validation**: Express Validator
